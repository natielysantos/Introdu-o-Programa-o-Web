CSS Zen Garden

The Beauty of CSS Design

Explore o poder dos estilos CSS para transformar o design sem alterar o conteúdo HTML. Este exemplo usa uma paleta inspirada em uma noite estrelada.

Introdução

Este projeto visa demonstrar como estilos podem influenciar completamente a estética de uma página sem a necessidade de alterar sua estrutura.

Você pode criar múltiplas variações visuais, utilizando apenas folhas de estilo diferentes.

Este é um exemplo inspirado no projeto CSS Zen Garden.

/* style.css */ /* 1. Aspectos gerais */ body { width: 960px; margin: 0 auto; font-family: sans-serif; background-color: #0D1B2A; /* Fundo escuro */ color: #E0E1DD; /* Fonte clara */ } p { text-indent: 1em; } /* 2. Títulos */ h1, h2, h3 { text-align: center; color: #FF6B6B; /* Cor de destaque */ } h1 { font-size: 1.8rem; } h2 { font-size: 1.4rem; } h3 { font-size: 1.2rem; } /* 3. Seções */ .resumo { text-align: center; } .principal p { line-height: 1.5; } /* 4. Abreviaturas */ abbr { font-variant: small-caps; }
